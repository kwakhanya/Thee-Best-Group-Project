Instruction for Single player

Key 'A : Move the Tetrimino left.
Key 'D : Move the Tetrimino right.
Key 'Z : Move the Tetrimino down faster (soft drop).
Key 'W : Rotate the Tetrimino 90 degrees clockwise.

Instruction for Two Player player

Player 1

Key 'A : Move the Tetrimino left.
Key 'D : Move the Tetrimino right.
Key 'Z : Move the Tetrimino down faster (soft drop).
Key 'W : Rotate the Tetrimino 90 degrees clockwise.

Player 2

Left Arrow Key: Move the Tetrimino left.
Right Arrow Key: Move the Tetrimino right.
Down Arrow Key: Move the Tetrimino down faster (soft drop).
Up Arrow Key: Rotate the Tetrimino 90 degrees clockwise.,
